MarkItDown

Powerful Document Conversion Tool

Key Features

Multi-Format Support

Convert PDF, Word, PowerPoint, Excel, and more

Smart Recognition

Built-in OCR and audio transcription

Easy to Use

Simple command-line interface

Download

Windows v1.0.0 64-bit macOS v1.0.0 Universal Linux v1.0.0 x86_64

Quick Start

markitdown input.pdf > output.md

Or use with pipe:

cat input.pdf | markitdown > output.md

Examples

Convert PDF

markitdown document.pdf > document.md

Convert Word

markitdown document.docx > document.md
